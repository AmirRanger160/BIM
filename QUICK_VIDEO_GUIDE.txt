🚀 نحوه استفاده:

برای اضافه کردن ویدیو:
1. به /admin/videos بروید
2. "➕ ویدیوی جدید" کلیک کنید
3. فرم را تکمیل کنید
4. لینک ویدیو را وارد کنید (YouTube/Vimeo/Direct URL)
5. ذخیره کنید

برای پخش ویدیو:
1. صفحه اصلی باز کنید
2. دکمه "▶ تماشای ویدیو" کلیک کنید
3. Modal باز شود و ویدیو پخش می‌شود

🔧 API Examples:

دریافت ویدیوها:
GET /api/videos?active_only=true&limit=10

دریافت ویدیوی خاص:
GET /api/videos/1

ایجاد ویدیو:
POST /api/videos
{
  "title": "عنوان",
  "description": "توضیح",
  "video_url": "https://youtube.com/...",
  "active": true
}

فعال/غیرفعال:
POST /api/videos/1/toggle

حذف:
DELETE /api/videos/1
